# Setup with Next.js - Flowbite React

<p className="text-lg">Copy the starter or follow the steps below to get started with Flowbite React in Next.js.</p>

---

## Flowbite Next.js Starter

- [GitHub Repo](https://github.com/tulupinc/flowbite-next-starter)
- [Demo](https://flowbite-next-starter.vercel.app/)

## Add Flowbite React to your Next.js project

1. Install dependencies:

```bash
npm install --save autoprefixer postcss tailwindcss flowbite flowbite-react
```

2. Create `postcss.config.js`:

```javascript
module.exports = {
  plugins: {
    autoprefixer: {},
    tailwindcss: {},
  },
};
```

3. Create `tailwind.config.js`:

```javascript
/**
 * @type {import('@types/tailwindcss/tailwind-config').TailwindConfig}
 */
module.exports = {
  content: ['./node_modules/flowbite-react/**/*.js', './pages/**/*.{ts,tsx}', './public/**/*.html'],
  plugins: [require('flowbite/plugin')],
  theme: {},
};
```

4. And replace the contents of `styles/globals.css` by:

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

5. Start using `flowbite-react`!

```jsx
'use client';

import { Alert } from 'flowbite-react';

export default function MyPage() {
  return <Alert color="info">Alert!</Alert>;
}
```
